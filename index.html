Integration eines kommerziellen Sprachassistenten in eine eigene Audioanwendung

Jiani Qu, Artur Solomonik

Amazon Alexa: Skill Entwicklung

  • Skill: Personalisierte Sprachanwendung
  • Hauseigenes öffentliches Toolkit für Entwicklung (Alexa Interaction Model)
  • Amazon bietet Deployment über Amazon Web Services an
    • Deployment über eigene Server möglich

Amazon Alexa: Interaction Model

  • Benutzeranfragen (Intents) als Interaktionsgrundlage
    • Beispielausdrücke (Sample Utterance)
    • Lexikon für Satzbestandteil eines speziellen Typs (Slots)
      • Vor- oder Benutzerdefiniert
    • Dialog Model für Verständnisprobleme

Amazon Alexa: Interaktionsprozess

  1. Öffnung einer Sitzung
  2. Amazon Echo schickt Stimmaufnahmen an Cloud-Service
  3. Umwandlung der Audioaufnahme in Text
  4. Abgleich mit passender Sample Utterances
    • Ähnlichkeit/Zugehörigkeit zu definierten Slots führt zu Lernprozess
  5. Passende Aktion am Nutzerendpunkt durchgeführt
    • Benutzerdefinierte Antwort: Text-To-Speech Umwandlung der definierten Antwort
